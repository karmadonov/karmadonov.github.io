## Overview

This is a static personal website hosted on GitHub Pages at `karma.dev`. It consists of a single `index.html` with inline CSS — no build step, no dependencies, no framework.

## Deployment

Changes pushed to `master` are automatically deployed via GitHub Pages. The `CNAME` file maps the custom domain `karma.dev` to the GitHub Pages host.

## Development

Open `index.html` directly in a browser to preview changes — no local server needed.
